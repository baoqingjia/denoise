DESCRIPTION:
A freeware (see licence terms below) programme for analysis of NMR Data, primarily FG-NMR (aka DOSY) data.

SYSTEM REQUIREMENT:
MATLAB 7.6 or above. Preferred but not required is the Optimisation Toolboxes. For PARAFAC analysis and for
enhanced performance (optional)


INSTALLATION:
Unzip DOSYToolbox_XXX.zip to your preferred directory.
Add the directory (with subdirectories) to the MATLAB path.
Type "DOSYToolbox" to start the graphical user interface.
